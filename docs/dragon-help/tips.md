# 指令卡片模板:
```html
<div class="command-info">
    <span class="command-name"></span>
    <span class="command-form">格式:</span>
    <span class="command-form-ins"></span>
    <span class="command-return">返回信息:</span>
    <span class="command-return-ins">

    </span>
</div>
```
```html
<div class="command-info">
    <span class="command-name">发公告</span>
    <span class="command-form">格式:</span>
    <span class="command-form-ins">
        发公告
        <span class="variable">(群号)</span>
    </span>
    <span class="command-return">返回信息:</span>
    <span class="command-return-ins">
        <span class="special-return">特殊信息:</span>
    </span>
</div>
```
```html
<div class="command-info">
    <span class="command-name">二次元的我</span>
    <span class="command-form">格式:</span>
    <span class="command-form-ins">
        /二次元的我
    </span>
    <span class="command-return">返回信息:</span>
    <span class="command-return-ins">
        二次元少女的<span class="variable">(nickname),(随机形容词)</span>
    </span>
</div>
```
